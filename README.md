## Comenzando 🚀

_Crea una carpeta nueva para el proyecto y Clona el repositorio_

```
git clone https://github.com/igorariza/course_app.git
```

Mira **Deployment** para conocer como desplegar el proyecto.


### Pre-requisitos 📋

_Java_
_Firebase_
_MySql_

```
java --version
firebase --version
mysql --version
```
